# 📋 变更日志

## [1.0.0] — 2026-05-23

### MVP 发布 🎉

完成了从 0 到 1 的核心功能建设，这是一个**完整可用的个人思维碎片工具**。

### 核心功能
- **碎片记录** — 支持短句记录，自动识别情绪/工作/阅读/习惯/决策 5 类标签
- **AI 洞察** — 长文本（>100 字）自动调用 AI 提取洞察和行动项
- **时间追踪** — 解析 `0130-0800 睡觉` 格式，自动计算时长和分类统计
- **日总结** — `今天总结` 命令 → 当日碎片 + 时间分布汇总
- **周复盘** — `周复盘` 命令 → 3+1 板块完整分析（情绪/主题/时间/愿景）
- **月复盘** — `月复盘` 命令 → 信念演变 + 模式识别
- **历史浏览** — `历史` 命令 → 按日期浏览全部记录
- **PWA 支持** — manifest.json + Service Worker，可添加到手机桌面
- **多模型支持** — 接入 DeepSeek / 千问 / 豆包，可自定义 API Endpoint
- **深色主题** — 护眼深色 UI，移动端适配

### 技术细节
- 零依赖纯前端，单个 HTML 文件（34 KB）
- IndexedDB 本地存储，数据用户完全掌控
- 响应式设计，手机/桌面均可使用
- Git 仓库初始化，GitHub 远程已配置
- 一键部署脚本 (deploy.bat)
