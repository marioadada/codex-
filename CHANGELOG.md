# 📋 变更日志



## [1.0.2] — 2026-05-23

### Bug 修复
- **刷新丢数据** — `init()` 现在会从 IndexedDB 加载最近 50 条记录并显示在聊天中，刷新页面内容不丢失
- **Markdown 渲染失效** — 移除 AI 结果中的 `.replace(/\n/g, '<br>')`，让 `renderMarkdown()` 直接处理原始换行，标题/列表等块级语法正确生效
- **标签分隔** — 标签 `<div>` 与内容之间加 `\n` 分隔，避免标题检测失效
- **短消息美化** — 短文本不再包裹 `<p>` 标签，减少不必要的间距
## [1.0.1] — 2026-05-23

### 改进
- **Markdown 渲染** — AI 回复中的 `**粗体**`、`##标题`、`` `代码` `` 等 Markdown 语法现在会正确渲染为格式化文本
- **视觉升级** — 新增标题、列表、引用、代码块、链接等元素的暗色主题样式
- **文档体系** — 新增 README.md / CHANGELOG.md / docs/roadmap.md

### 技术细节
- 新增 `renderMarkdown()` 函数，支持：h3/h4 标题、粗体/斜体、行内代码、代码块、引用、有序/无序列表、链接、分隔线
- 在 `addMessage()` 中对 `bot` 类型自动应用 Markdown 渲染
- 新增 15 条 CSS 规则覆盖所有渲染元素

## [1.0.0] — 2026-05-23

### MVP 发布 🎉

完成了从 0 到 1 的核心功能建设，这是一个**完整可用的个人思维碎片工具**。

### 核心功能
- **碎片记录** — 支持短句记录，自动识别情绪/工作/阅读/习惯/决策 5 类标签
- **AI 洞察** — 长文本（>100 字）自动调用 AI 提取洞察和行动项
- **时间追踪** — 解析 `0130-0800 睡觉` 格式，自动计算时长和分类统计
- **日总结** — `今天总结` 命令 → 当日碎片 + 时间分布汇总
- **周复盘** — `周复盘` 命令 → 3+1 板块完整分析（情绪/主题/时间/愿景）
- **月复盘** — `月复盘` 命令 → 信念演变 + 模式识别
- **历史浏览** — `历史` 命令 → 按日期浏览全部记录
- **PWA 支持** — manifest.json + Service Worker，可添加到手机桌面
- **多模型支持** — 接入 DeepSeek / 千问 / 豆包，可自定义 API Endpoint
- **深色主题** — 护眼深色 UI，移动端适配

### 技术细节
- 零依赖纯前端，单个 HTML 文件（34 KB）
- IndexedDB 本地存储，数据用户完全掌控
- 响应式设计，手机/桌面均可使用
- Git 仓库初始化，GitHub 远程已配置
- 一键部署脚本 (deploy.bat)
