# 🧩 思维碎片 (Fragment Tracker)

> 一个轻量级「思维碎片」记录工具。记录日常碎碎念、时间开销，让 AI 帮你做周复盘、月复盘。

**在线地址：** https://marioadada.github.io/codex-/

---

## ✨ 功能一览

| 输入 | 行为 |
|------|------|
| `好累`（<30 字短句） | 📝 记录碎片 + 自动识别情绪标签 |
| `今天工作好烦，老板又改需求...`（>100 字） | 🤖 调 AI 分析 → 输出洞察 + 行动项 |
| `0130-0800 睡觉`（时间格式） | ⏱ 自动计算时长、分类、累计统计 |
| `今天总结` | 📅 汇总今日碎片 + 时间分布饼图 |
| `周复盘` | 📊 完整 3+1 板块分析（含愿景对齐） |
| `月复盘` | 📈 信念演变 + 行为模式识别 |
| `历史` / `history` | 📚 浏览所有历史碎片，按日期筛选 |

## 🏗 技术架构

```
fragment-tracker/
├── index.html        ← 单页应用（全部功能自包含）
├── manifest.json     ← PWA 配置（添加到手机桌面）
├── sw.js             ← Service Worker（离线缓存）
├── deploy.bat        ← 一键部署脚本
├── README.md         ← 本文档
├── CHANGELOG.md      ← 变更日志
└── docs/
    └── roadmap.md    ← 开发路线图
```

### 技术栈
- **纯前端** — 零依赖，零构建，一个 HTML 搞定
- **存储** — IndexedDB（浏览器本地存储，不占服务器）
- **AI** — 可选接入 DeepSeek / 千问 / 豆包 API
- **PWA** — 支持添加到手机桌面，离线可用

### 数据存储
所有数据存在你的浏览器 IndexedDB 中，有两个表：
- `fragments` — 碎片记录（内容、标签、时间、AI 分析结果）
- `settings` — 配置（API Key、模型选择等）

> ⚠️ 数据在浏览器本地，清除浏览器数据会丢失。建议定期使用「历史」功能导出。

## 🔑 配置 AI

### 支持的 AI 服务

| 服务 | 默认模型 | 免费额度 |
|------|---------|---------|
| **DeepSeek** | `deepseek-chat` | 注册送 500 万 tokens |
| **千问 (通义千问)** | `qwen-max` | 新用户送 200 万 tokens |
| **豆包** | 需在控制台确认 | 实名送体验额度 |

### 配置步骤
1. 打开应用
2. 右上角 ⚙️ 设置
3. 选择 **DeepSeek**（或其他）
4. 填入你的 **API Key**
5. 点击 **保存**

> 💡 即使不配 AI，手动记录碎片、时间追踪、查看历史等离线功能全部可用。

## 📖 使用指南

### 碎片记录
直接在底部输入框打字发送即可。支持自动标签：
- 😢 **情绪** — `烦` `累` `开心` `焦虑` 等情感词汇
- 💼 **工作** — `工作` `项目` `方案` `代码` 等
- 📚 **阅读** — `书` `文章` `学习` 等
- 🔄 **习惯** — `运动` `冥想` `早睡` 等
- 💡 **决策** — `决定` `选择` `计划` 等

### 时间记录
格式：`开始时间-结束时间 活动描述`
```
0130-0800 睡觉
0915-1027 写方案
1400-1430 午休
```
自动计算时长，按活动内容分类，累积统计。

### 复盘命令
```
今天总结    → 今日碎片汇总 + 时间分布
周复盘      → 本周完整分析（情绪/主题/时间/愿景）
月复盘      → 本月模式识别 + 信念演变
历史        → 浏览全部历史记录
```
